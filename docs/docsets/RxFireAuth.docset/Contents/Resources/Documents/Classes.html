The following classes are available globally.

  • SignInWithAppleHandler

    A helper class that handles the flow of Sign in with Apple.

    An instance of this class is automatically created and invoked by UserManager when you ask it to signInWithApple(in:updateUserDisplayName:allowMigration:). You can use it also without a user manager associated.

    Sign in with Apple is only available on iOS 13 and macOS 10.15 or later.

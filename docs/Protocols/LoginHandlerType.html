LoginHandlerType

public protocol LoginHandlerType

This protocol identifies a login handler object that is used by the library to authenticate with a 3rd-party provider, such as Apple or Google.

You will get an instance of this protocol when reading the value of loginHandler in implementations of IUserManager- You can use it to redirect incoming calls from the system browser, for example, when authenticating with a OAuth provider that redirects directly to your app (such as Google Sign In).
